# SQL Queries Assignment

## 1. Days Remaining in Current Year
```sql
SELECT DATEDIFF(
    STR_TO_DATE(CONCAT(YEAR(CURDATE()), '-12-31'), '%Y-%m-%d'),
    CURDATE()
) AS DAYS_REMAINING;
```

## 2. Highest, Lowest Salary and Difference
```sql
SELECT 
    MAX(SAL) AS HIGHEST_SALARY,
    MIN(SAL) AS LOWEST_SALARY,
    MAX(SAL) - MIN(SAL) AS DIFFERENCE
FROM EMPLOYEE;
```

## 3. Employees with Commission > 25% of Salary
```sql
SELECT ENAME
FROM EMPLOYEE
WHERE COMM > 0.25 * SAL;
```

## 4. Salary in Dollar Format
```sql
SELECT ENAME, CONCAT('$', SAL) AS SALARY_IN_DOLLARS
FROM EMPLOYEE;
```

## 5. Matrix Query (Job-wise Salary by Department)
```sql
SELECT JOB,
SUM(CASE WHEN DEPTNO = 10 THEN SAL ELSE 0 END) AS DEPT10,
SUM(CASE WHEN DEPTNO = 20 THEN SAL ELSE 0 END) AS DEPT20,
SUM(CASE WHEN DEPTNO = 30 THEN SAL ELSE 0 END) AS DEPT30,
SUM(SAL) AS TOTAL
FROM EMPLOYEE
GROUP BY JOB;
```

## 6. Total Employees and Year-wise Hiring Count
```sql
SELECT 
COUNT(*) AS TOTAL_EMPLOYEES,
SUM(YEAR(HIREDATE) = 1980) AS Y1980,
SUM(YEAR(HIREDATE) = 1981) AS Y1981,
SUM(YEAR(HIREDATE) = 1982) AS Y1982,
SUM(YEAR(HIREDATE) = 1983) AS Y1983
FROM EMPLOYEE;
```

## 7. Last Sunday of Any Month
```sql
SELECT LAST_DAY(CURDATE()) - INTERVAL (DAYOFWEEK(LAST_DAY(CURDATE())) - 1) DAY AS LAST_SUNDAY;
```

## 8. Department-wise Employee Count
```sql
SELECT DEPTNO, COUNT(*) AS TOTAL_EMPLOYEES
FROM EMPLOYEE
GROUP BY DEPTNO;
```

## 9. Job-wise Employee Count
```sql
SELECT JOB, COUNT(*) AS TOTAL_EMPLOYEES
FROM EMPLOYEE
GROUP BY JOB;
```

## 10. Department-wise Total Salary
```sql
SELECT DEPTNO, SUM(SAL) AS TOTAL_SALARY
FROM EMPLOYEE
GROUP BY DEPTNO;
```
